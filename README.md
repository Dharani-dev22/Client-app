# Jay Balaji Super Market 🛒

A full-stack Android application developed as a college project to streamline the shopping experience for Jay Balaji Super Market. This app provides a seamless interface for customers to browse products and manage orders.

## 📥 Download APK
You can download the latest stable version of the app to install on your Android device from the link below:

**[Download Jay Balaji Super Market v1.0](https://github.com/Dharani-dev22/Client-app/releases/tag/v1.0)**

> **Note:** Since this is a direct APK install, you may need to enable "Install from Unknown Sources" in your Android settings.

---

## ✨ Key Features
- **Product Catalog:** Browse a wide variety of supermarket items categorized for ease of use.
- **Secure Payments:** Integrated with **Razorpay** for safe and fast transactions.
- **Real-time Database:** Powered by **Firebase** for inventory management and user data.
- **Admin Dashboard:** Specific interface for store management to update stock and view orders.
- **Modern UI:** Built with **Jetpack Compose** for a smooth, responsive user experience.

## 🛠 Tech Stack
- **Language:** Kotlin
- **UI Framework:** Jetpack Compose
- **Backend/Database:** Firebase (Auth, Firestore)
- **Payment Gateway:** Razorpay SDK
- **Architecture:** MVVM (Model-View-ViewModel)
